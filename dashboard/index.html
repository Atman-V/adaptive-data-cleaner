What It Does

Discover what the cleaning pipeline can do

The engine reads and profiles your spreadsheet column by column, then applies a smart sequence of data-cleansing operations to prepare your file for analysis.

🗂️

Table Structure Detection

Detects the header row, delimiter (comma/tab/semicolon/pipe), and encoding for CSV. For Excel, it unmerges merged cells, folds multi-level headers into one, and splits multiple stacked tables on a sheet — analyzing the largest block.

🧬

Type & Schema Normalization

Infers the right data type for every column — text, number, date, email, phone, categorical — using statistical and pattern-based inference.

🛡️

Data Validation & Fixes

Audits every column for missing values, mixed date formats, invalid emails, and numeric outliers — then applies the fix rule you choose.

🧩

Duplicate & Anomaly Detection

Flags fully duplicate rows, near-duplicate rows that only differ by case or whitespace (e.g. "John Smith" vs "john smith "), primary-key collisions, IQR-based numeric outliers, and out-of-range negative values.
